## 项目构筑 & 运行提示 Tips

0. 在mysql数据库中创建tflz数据库

1. 修改pom.xml、application.propoties文件，将账号、密码和端口换成你的

2. 在使用npm install Vue-cli项目后，不要忘了移动map文件夹（具体进入tflz-front文件夹的README.md文档查看）
